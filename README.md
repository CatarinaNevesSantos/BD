# 🗄️ Intelligent Vending Machines - Database Project

Modelling and implementation of a relational database for managing
Intelligent Vending Machines (IVM) in retail environments.
The system manages smart vending machines installed at retail points,
handling products, categories, shelves, planograms, retailers and
replenishment events.

## 🗂️ Deliverables

- **Part 1** - Entity-Relationship (ER) diagram and integrity constraints
- **Part 2** - Relational model, relational algebra and SQL queries
- **Part 3** - SQL, triggers, web application (Python CGI) and OLAP queries

## 🛠️ Technologies

- SQL
- Python (CGI scripts)
- HTML/CSS
